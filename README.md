# 履歴書作成ポータルサイト

履歴書作成と自己PR文生成をサポートするポータルサイトです。

## 機能

### 1. 履歴書作成
外部サイトへのリンクから、プロフェッショナルな履歴書を作成できます。

### 2. 自己PR生成AI
AIチャット機能を使用して、効果的な自己PR文を作成できます。

## ファイル構成

```
ResumePortal/
├── index.html          # メインページ（ポータル）
├── ai-chat.html        # 自己PR生成AIページ
├── css/
│   ├── style.css       # メインページのスタイル
│   └── ai-chat.css     # AIチャットページのスタイル
├── js/
│   └── ai-chat.js      # AIチャット機能のロジック（モックAPI）
└── README.md           # プロジェクト説明
```

## 使い方

1. `index.html`をブラウザで開いてください
2. 「履歴書作成」または「自己PR生成AI」を選択してください
3. 自己PR生成AIを使用する場合は、利用規約と個人情報保護方針に同意してください
4. チャット欄にメッセージを入力して、AIと対話してください

## 技術仕様

- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなデザインとレスポンシブ対応
- **JavaScript (ES6+)**: インタラクティブな機能とモックAPI
- **ローカルストレージ**: 利用規約への同意状態を保存

## ブラウザ対応

モダンブラウザ（Chrome、Firefox、Safari、Edge）の最新版に対応しています。

## 注意事項

- 自己PR生成AIはモックAPIを使用しています（実際のAI APIは使用していません）
- チャット内容はブラウザのローカルストレージに保存される場合があります
- 外部サイトへのリンクは新しいタブで開きます
